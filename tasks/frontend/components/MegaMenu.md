# MegaMenu Redesign & Content Strategy

## Objective

Transform the existing Navbar dropdowns into "Artisan MegaMenus" that reflect the premium brand identity of DySalatto and DyCake. Moving away from standard lists to a highly professional, visually rich layout.

## Tasks

### 1. DYSALATTO (Bowl Menu)

- [ ] **Visual Identity:** Integrate a subtle grain texture and brand-terracotta accents.
- [ ] **Content Enrichment:**
  - **Signature Art:** High-protein and best-seller bowls.
  - **Seasonal Palette:** Monthly/seasonal special bowls.
  - **Artist's Choice:** Chef's special combinations.
- [ ] **Interactive Elements:** Add "Coming Soon" or "New" badges to specific items.
- [ ] **Hover State:** Refine item hover with a "spotlight" effect (subtle glow).

### 2. DYCAKE (Sweet Menu)

- [ ] **Visual Identity:** Use a warmer, "dark chocolate/sand" color palette.
- [ ] **Content Enrichment:**
  - **Guilt-Free Art:** Sugar-free and gluten-free cakes.
  - **Daily Masterpieces:** Freshly baked daily cakes (San Sebastian, etc.).
- [ ] **Iconography:** Add minimal custom icons for "Gluten-Free" or "Vegan" tags.

### 3. Professional Polish (Global)

- [ ] **Fluid Entrance:** Implement a "staggered fade-in" for menu categories.
- [ ] **Backdrop Blur:** Increase to `backdrop-blur-xl` for better contrast against the background.
- [ ] **Glassmorphism:** Refine the border and shadow of the dropdown container.
- [ ] **Responsive check:** Ensure the MegaMenu doesn't break on smaller desktop screens (laptop size).
