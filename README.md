## ✨ Özellikler

- **🎨 Modern Arayüz:** Glassmorphism ve akıcı animasyonlarla zenginleştirilmiş kullanıcı deneyimi.
- **⚡ Yüksek Performans:** Next.js 16 ve React 19 ile en yeni web teknolojileri.
- **📱 Mobil Uyumlu:** Tüm cihazlarda kusursuz görünen responsive tasarım.
- **🛠️ Güçlü Backend:** Node.js, Express ve Prisma (PostgreSQL) ile güvenilir veri yönetimi.
- **🎯 Mega Menu:** Kategori bazlı, detaylı ve görsel olarak zenginleştirilmiş menü yapısı.

## 🛠️ Teknoloji Yığını

### Frontend
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Animation:** Framer Motion & Lucide Icons
- **State Management:** React Hooks

### Backend
- **Runtime:** Node.js
- **Framework:** Express
- **ORM:** Prisma
- **Database:** PostgreSQL

## 🚀 Başlangıç

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları takip edin:

### 1. Depoyu Klonlayın
```bash
git clone https://github.com/kullanici/dysalatto.git
cd DySalatto
```

### 2. Bağımlılıkları Kurun
```bash
# Frontend
cd frontend
npm install

# Backend
cd ../backend
npm install
```

### 3. Çevresel Değişkenleri Ayarlayın
Kök dizindeki `.env` dosyasını referans alarak `frontend/.env` ve `backend/.env` dosyalarını oluşturun.

### 4. Geliştirme Sunucusunu Başlatın
```bash
# Frontend için
cd frontend
npm run dev

# Backend için
cd backend
npm run dev
```
